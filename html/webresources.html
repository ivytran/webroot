Web Resources

  • w3 Schools | 5/5
    This website is great for learning web development as it gives clear examples that can be edited in a web browser. They also offer help outside of web development with lessons in Python and Java.

  • Html Dog | 4/5
    HTML dog provides web development tutorials in very small and easy to understand lessons. It can really help beginners learn how to code simple websites. However to a more experienced web developer this web doesent offer anything new.

  • Tutorial Republic | 5/5
    Tutorial Republic offers similar tutorials to w3 Schools but with a larger focus on HTML, CSS, and Javascript. It goes into more details at times and is similarly structured to HtmlDog in it's lessons. This site is good for both experienced and inexperienced web developers.
